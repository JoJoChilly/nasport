   # NASPort 

   Synology NAS API调用工具，用于访问和管理您的Synology NAS。

   ## 功能

   - 登录到Synology NAS
   - 调用Synology API
   - 安全管理会话

   ## 配置

   1. 复制 `synology_config.template.py` 为 `synology_config.py`
   2. 在 `synology_config.py` 中填入您的NAS信息

   ## 使用方法

   1. 运行 `helloworld.py` 进行测试
   2. 根据需要修改 `helloworld.py` 中的API调用

   ## 依赖

   - requests
